入门OJ 4246： [Noip模拟题]中位数

发表于 2018-08-26 更新于 2020-09-24 分类于 OI

题目

Description

给出1~n的一个排列，统计该排列有多少个长度为奇数的连续子序列的中位数是b。 中位数是指把所有元素从小到大排列后，位于中间的数。

Input

第一行为两个正整数n(n<=100000)和b(1<=b<=n)，第二行为1~n 的排列。 ## Output 输出一个整数，即中位数为b的连续子序列个数。 ## Sample Input

## Sample Output 样例解释：{4}, {7,2,4}, {5,7,2,4,3}和{5,7,2,4,3,1,6}。 # 题解
